# Frequency Analysis of Words

The script return 10 most frequency words in a text

# System requirements

Python3.5

# Quickstart
python lang_frequency.py path_to_text
```bash
$ python lang_frequency.py text.txt
the
of
a
Lorem
and
Ipsum
in
is
it
has
```
